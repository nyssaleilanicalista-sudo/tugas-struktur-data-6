# 🎬 Queue Visualizer (Python GUI)

Aplikasi ini merupakan visualisasi interaktif dari konsep **Queue (Antrian)** dalam struktur data menggunakan Python.
Program menampilkan berbagai studi kasus nyata dalam bentuk animasi sederhana agar mudah dipahami.

---

## 📌 Fitur Utama

* 🔁 Visualisasi konsep **FIFO (First In First Out)**
* 🎮 5 kasus implementasi queue:

  1. Printer Queue
  2. Hot Potato
  3. Priority Queue (Rumah Sakit)
  4. BFS (Breadth-First Search)
  5. Simulasi Bandara
* 🎛️ Kontrol interaktif:

  * Enqueue (tambah data)
  * Dequeue (hapus data)
  * Start / Stop animasi
  * Reset
* 📊 Statistik real-time (jumlah antrian, processed, dll)
* 🖥️ GUI sederhana menggunakan Tkinter (tanpa install tambahan)

---

## 🧠 Konsep yang Digunakan

### 1. Queue (Antrian)

Struktur data linear dengan prinsip:

> **First In, First Out (FIFO)**

### 2. Priority Queue

Antrian dengan prioritas tertentu (misalnya pasien kritis didahulukan)

### 3. Circular Queue (Hot Potato)

Simulasi antrian melingkar

### 4. BFS (Breadth-First Search)

Algoritma traversal graph berbasis queue

### 5. Simulasi Sistem Nyata

Contoh:

* Antrian printer
* Loket bandara

---

## 🚀 Cara Menjalankan

1. Pastikan Python sudah terinstall (disarankan Python 3.10+)

2. Jalankan file:

```bash
python main.py
```

3. Gunakan tombol di aplikasi:

* Pilih kasus (Printer, Hot, RS, BFS, Bandara)
* Klik **Start** untuk animasi
* Gunakan **Enqueue / Dequeue** untuk interaksi
